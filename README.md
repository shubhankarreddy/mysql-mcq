# MySQL MCQ Skills Assessment

A practical MySQL multiple-choice quiz webpage covering Easy (S1), Moderate (S2), Hard (S3), and Query Challenge (QC) levels.

Live site: https://shubhankarreddy.github.io/mysql-mcq/

## Contents

- 25 questions per level
- Industry-relevant MySQL scenarios
- Immediate feedback with explanations
- Progress tracking and topic breakdown

## Levels

- Easy (S1): SQL command categories, data types, WHERE, aggregation, strings, ordering, limits, and dates
- Moderate (S2): joins, subqueries, CTEs, views, stored procedures, indexes, CASE/IF, set operations, and cross joins
- Hard (S3): recursive CTEs, window functions, ranking, moving averages, and ETL patterns
- Query Challenge (QC): practical end-to-end query selection problems
